### A Bucket for [Scoop Installer](https://scoop.sh)

To start with:
```
scoop bucket add maoyeedy_scoop https://github.com/maoyeedy/scoop
```
To Install a manifest from [bucket folder](bucket/):
```
scoop install $manifest
```
