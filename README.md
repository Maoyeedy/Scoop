### Yeedy's bucket for [Scoop Installer](https://scoop.sh)

To add this bucket:
```
scoop bucket add maoyeedy_scoop https://github.com/maoyeedy/scoop
```
To install:
```
scoop install processing4
scoop install eudic
scoop install ...
```
