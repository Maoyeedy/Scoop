### Yeedy's buckets for [Scoop Installer](https://scoop.sh)

To add this bucket:
```powershell
scoop bucket add maoyeedy_scoop https://github.com/maoyeedy/scoop
```
To install:
```powershell
scoop install processing4
scoop install eudic
scoop install ...
```
